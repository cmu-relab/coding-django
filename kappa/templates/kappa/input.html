Inter-rater Reliability Input File Format

The input data for computing inter-rater reliability on nominal ratings must be presented as a comma-separated values (CSV) formatted file: the first or header row must be your rater labels or names, the first column must be the item labels or names, and then each cell X, Y for row X ≥ 1 and column Y ≥ 1 must be the rating or code provided by rater X for item Y.

For example, the HTML version of an example table might look something like this for five raters, three items numbered 1-3 and the codes A, B, C and D:

  Rater 1 Rater 2 Rater 3 Rater 4 Rater 5
1 A A B C A
2 C B C C C
3 D D D D A

To generate a blank input file, select the number of raters and items below and you will be prompted to download the file.
